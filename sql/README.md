# SQL Analysis

This folder contains SQL queries used to validate key business metrics and support the analysis performed in the dashboard.

SQL was used to confirm calculations and ensure the accuracy of insights presented in the Power BI dashboard.

The analysis includes:

- Total sales and product distribution calculations
- Category-level performance analysis
- Discount distribution across product categories
- Customer rating analysis
- Validation of calculated metrics such as value score

SQL queries help ensure data accuracy and strengthen the reliability of the insights generated.
