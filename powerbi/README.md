## Power BI Dashboard

The Power BI dashboard visualizes retail business performance and pricing strategies.

 Main dashboard sections:

 # Sales Overview
 
• Total Sales

• Total Products

• Average Discount Rate

• Average Customer Rating


Category Performance

• Product distribution by category

• Sales contribution by category


Discount Analysis

• Discount strategy across categories


Value Score Analysis

• Value score (rating per price unit)

• Identifies value-for-money products


Rating vs Price Analysis

• Relationship between product ratings and pricing


Drill-through Analysis

• Product-level analysis for deeper insights
