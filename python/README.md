# Python Data Preparation

This folder contains the Python scripts used for data preparation and cleaning.

Python was used to process the raw retail data before loading it into SQL and Power BI.

Key tasks performed in Python:

- Importing and combining multiple CSV files into a single dataset
- Handling missing values and inconsistent records
- Converting and standardizing data types
- Cleaning and formatting columns for analysis
- Preparing the dataset for SQL validation and dashboard visualization
